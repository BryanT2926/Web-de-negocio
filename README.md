# 🍽️ La Fogata Dorada - Página Web

Sitio web responsive para un restaurante llamado **La Fogata Dorada**, diseñado para mostrar el menú, servicios y facilitar el contacto con clientes.

## 🚀 Características

- Diseño moderno y responsive (adaptable a móviles)
- Sección de menú del restaurante
- Información del negocio
- Botón de contacto (WhatsApp / formulario)
- Navegación sencilla e intuitiva

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
